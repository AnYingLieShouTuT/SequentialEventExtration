# TravelSequentialEvent
Sequential Event Experiment based on Travel note crawled from XieCheng，基于50W携程出行游记的采集，顺承事件实验

# 项目构成  
本项目由两个部分的组成．  
１，出行领域语料的获取  
２，基于出行语料的事件挖掘  

# 出行领域语料的获取
1) 语料来源:携程出行游记  
2) 时间范围:2018年7月14日之前  
3) 采集方式:使用scrapy编写爬虫脚本进行抓取  
4) 采集规模:共采集505767篇，量级50W　　
5) 采集脚本目录:news_spider/travelspider
